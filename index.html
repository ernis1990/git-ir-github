Git

Git'as populiariausia kodo versijavimo programa

  • git init - sukuria gito repoziotrija dabartiniame aplankale
  • git status - nurodo informacija apie lokalios gito repozitorijos statusa
  • git add - prideda nurodytus failus prie komitui paruostu failu(stage) (git add . - paima visus failus)
  • git rm --cached - isima nurodytus failus is stagenimo zonos
  • git commit -m "zinute"
  • git restore - grazina failus i buvusio komito busena(panaikina naujus pakitimus po buvusio paskutinio komito)
  • git push - is lokalio repozitorijos nusuncia komitus i isorine repozitorija
  • pull - is isorines repozitorijos atsiunciamas comitas i lokalia repozitorija
  • cole - klonuoja nuoroda repozitorija is isorines repozitorijos
